# Hrdlr Arduboy

![](/banner.png)

``` sh
# Install platformio
brew install platformio

# Install arduboy lib (version 1.1.1)
platformio lib install 192

# Build
make
```

## Arduboy

Arduboy is based on Arduino, an open source development platform for
programming microcontrollers.

- 8bit Micro
- 32u4 ATMEga
- 32kb flash
- 2.5kb RAM
- 1kb EEPROM
- 16mhz freq
- 1bit Display
- 128x64px OLED
